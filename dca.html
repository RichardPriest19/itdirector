DCA

Docker Certified Associate – Mirantis certification for Docker container platform administration and deployment

Overview

The Docker Certified Associate (DCA) is awarded by Mirantis (current steward of Docker Enterprise) and validates competency in containerisation using Docker and Docker Enterprise. It covers container lifecycle management, image creation, networking, storage, security, and orchestration with Docker Swarm, making it the standard certification for professionals deploying and managing containerised workloads in enterprise environments.

Key Value: DCA is the definitive professional standard for Docker platform administrators — demonstrating hands-on competency in enterprise container operations, security hardening, and orchestration that organisations depend on for reliable, scalable containerised infrastructure.

Core Exam Domains

Container Orchestration

  • Deploy and manage services in Docker Swarm — nodes, stacks, services, and replicas
  • Configure service scaling, rolling updates, and rollbacks in Swarm mode
  • Apply resource constraints, placement constraints, and service labels
  • Understand Kubernetes orchestration concepts in the context of Mirantis Kubernetes Engine (MKE)

Image Creation, Management & Registry

  • Write and optimise Dockerfiles using multi-stage builds and layer caching
  • Tag, push, pull, and manage images in Docker Hub and private registries
  • Implement image signing and content trust using Docker Notary
  • Scan images for vulnerabilities using Docker Scout and registry scanning tools

Installation, Configuration & Security

  • Install and configure Docker Engine on Linux systems with appropriate storage and logging drivers
  • Implement Docker security hardening — user namespaces, seccomp profiles, and AppArmor/SELinux
  • Configure Docker Content Trust, image signing, and role-based access control
  • Manage TLS certificates for Docker API and registry security

Networking & Storage

  • Configure and manage Docker network drivers — bridge, overlay, host, and macvlan
  • Implement container DNS, service discovery, and network segmentation in Swarm
  • Manage persistent storage using Docker volumes, bind mounts, and volume plugins
  • Troubleshoot container networking and connectivity issues

Exam Details

  • 55 multiple-choice and discrete option questions over 90 minutes
  • Recommended prerequisite: 6+ months of hands-on Docker experience
  • Certification valid for 2 years; renewed by retaking the DCA examination
  • Complements CKA (Certified Kubernetes Administrator) for container platform professionals

IT Director Responsibilities

  • Require DCA for DevOps engineers, platform engineers, and SREs managing Docker and container infrastructure
  • Use DCA as the baseline qualification before advancing staff to CKA for Kubernetes environments
  • Ensure Docker deployments are administered by DCA-qualified staff to maintain security and operational standards
  • Integrate container security hardening practices from DCA into the DevSecOps pipeline
