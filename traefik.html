Traefik

Modern HTTP reverse proxy and load balancer designed for dynamic containerized and microservices environments

Overview

Traefik automatically discovers services running in Docker, Kubernetes, Consul, and other orchestrators, and configures routing rules dynamically without manual intervention.

Key Features

  • Automatic service discovery and configuration
  • Built-in middleware for authentication, rate limiting, and retries
  • Native support for Let's Encrypt TLS
  • Excellent observability with metrics and tracing
