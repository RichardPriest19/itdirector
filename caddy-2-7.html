Caddy 2.7

Modern, automatic HTTPS web server with simple configuration, built-in security, and excellent developer experience

Overview

Caddy is a lightweight, production-ready web server that automatically provisions and renews TLS certificates via Let's Encrypt and uses a human-readable Caddyfile configuration.

Key Advantages

  • Automatic HTTPS by default
  • Simple, declarative configuration
  • Built-in reverse proxy and load balancing
  • Excellent support for modern web applications
