ISO/IEC 23009

ISO/IEC 23009 – Dynamic adaptive streaming over HTTP (DASH) family of standards (2022 core edition)

Overview

ISO/IEC 23009 (MPEG-DASH) defines formats and protocols for **Dynamic Adaptive Streaming over HTTP**, enabling adaptive bitrate streaming of media content over standard HTTP. The core standard is **ISO/IEC 23009-1:2022** (5th edition), which specifies the Media Presentation Description (MPD) XML format, segment formats, and client/server behavior for adaptive streaming. In March 2026, ISO/IEC 23009 remains the dominant open standard for OTT video, live streaming, and low-latency applications (sports, gaming, e-learning), used by major platforms (Netflix, YouTube, Disney+, broadcasters), CDNs, and device ecosystems. Supporting parts include profiles (23009-2), client requirements (23009-3), segment formats (23009-4), server assistance (23009-5), ad insertion (23009-6), common encryption (23009-7), and low-latency extensions. It is interoperable with HLS (Apple), CMAF (Common Media Application Format), and 5G media delivery.

Key Value: ISO/IEC 23009 (DASH) enables high-quality, adaptive, HTTP-based video streaming with open interoperability, low-latency options, multi-CDN support, ad insertion, and DRM compatibility — reducing costs, improving viewer experience, and ensuring broad device compatibility across OTT, broadcast, and enterprise video use cases.

Core Parts & Features (2022/2026)

ISO/IEC 23009-1:2022 – Media Presentation & Segments

  • MPD (Media Presentation Description) XML format
  • Segment formats (ISO BMFF, MPEG-2 TS)
  • Adaptive bitrate switching & quality levels
  • Low-latency mode & chunked transfer

Supporting Parts

  • 23009-2 – Profiles (live, on-demand, low-latency)
  • 23009-3 – Client conformance & reference software
  • 23009-4 – Segment formats & CMAF integration
  • 23009-6 – Ad insertion & server-based ad decisioning
  • 23009-7 – Common encryption & DRM signaling
  • 23009-8 – Session-based DASH & timed metadata

Integration & Alignment

  • Compatible with HLS (Apple), CMAF, WebRTC
  • Supports 5G media delivery & edge computing
  • Aligns with MPEG Common Media Application Format (CMAF)

Key Adoption & Trends in 2026

Low-Latency & Live Streaming

  • Low-latency DASH (LL-DASH) maturity for sports/gaming
  • Sub-second latency with chunked CMAF
  • Server push & client pull optimizations

DRM & Security

  • Common encryption (CENC) & multi-DRM signaling
  • Integration with Widevine, PlayReady, FairPlay
  • Secure token & license acquisition

Enterprise & OTT

  • Multi-CDN switching & ABR heuristics
  • Personalized ad insertion & server-side stitching
  • 5G/edge delivery & immersive media support
