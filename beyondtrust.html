BeyondTrust (Bomgar)

Privileged access management and secure remote support platform

Overview

BeyondTrust is a comprehensive privileged access management (PAM) and secure remote support platform combining access control with session recording and monitoring. Originally known as Bomgar before acquisition by BeyondTrust, the platform is widely used by organizations requiring secure remote support with strict audit and compliance requirements. BeyondTrust focuses on PAM and remote access for sensitive environments including healthcare, finance, and government where privileged access control is critical.

Key Strength: BeyondTrust combines remote support with privileged access controls, providing both convenience and comprehensive security audit trails.

Core Components

Remote Support

  • Secure remote access to customer/user devices
  • Session recording and audit
  • End-to-end encryption
  • Unattended and on-demand support

Privileged Access Management

  • Manage privileged accounts
  • Just-in-time access requests
  • Privileged session monitoring
  • Credential vault

Session Recording

  • Record all support sessions
  • Playback and review
  • Compliance and audit trail

Key Features

Security Controls

  • Multi-factor authentication
  • Role-based access control
  • IP whitelisting
  • Custom security policies

Compliance Support

  • HIPAA compliance for healthcare
  • PCI-DSS for payment systems
  • SOX for finance
  • Audit logs and reporting

Administration

  • User and group management
  • Permission controls
  • Reporting and dashboards
