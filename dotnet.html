.NET

.NET – Microsoft unified, cross-platform development platform for cloud, web, mobile, desktop, AI & IoT

Overview

.NET is Microsoft's free, cross-platform, open-source developer platform for building high-performance, modern applications across cloud, web, desktop, mobile, gaming, IoT, and AI/ML. In March 2026, **.NET 9** (LTS, released Nov 2025) is the current production version, with **.NET 10** in preview. It features massive performance gains (JIT, Native AOT, Arm64), **cloud-native maturity** (minimal APIs, gRPC, .NET Aspire), **AI ecosystem** (ML.NET 3.x, Semantic Kernel, ONNX Runtime), **Blazor** full-stack web (WebAssembly + server), **.NET MAUI** cross-platform mobile/desktop, **EF Core 9** ORM, **C# 13** language features, and strong enterprise tooling (Azure SDKs, GitHub integration, Visual Studio 2025+). .NET runs everywhere — Windows, Linux, macOS, iOS, Android, browser — with unified tooling, runtime, and libraries.

Key Value: .NET delivers high-performance, cross-platform development with unified tooling, deep AI/ML integration, cloud-native excellence, and long-term support — enabling teams to build secure, scalable, modern applications faster, with lower total cost and fewer platform silos.

Core Capabilities (2026)

ASP.NET Core & Web

  • Minimal APIs, Razor Pages, MVC, Blazor (WebAssembly + Server)
  • gRPC, SignalR, Hot Reload, .NET Aspire for distributed apps
  • Blazor United (hybrid WebAssembly + Server rendering)

Cross-Platform & Mobile/Desktop

  • .NET MAUI – single codebase for iOS, Android, Windows, macOS
  • Native AOT for fast startup & small binaries
  • Windows Forms, WPF, WinUI 3 (Windows-only)

AI/ML & Data

  • ML.NET 3.x – end-to-end ML pipelines
  • Semantic Kernel – AI orchestration & plugins
  • EF Core 9 – high-performance ORM, JSON columns, AOT support

Cloud & DevOps

  • .NET Aspire – cloud-native app composition
  • Azure SDKs & GitHub Actions integration
  • Container-first (Docker, Kubernetes, OCI images)
