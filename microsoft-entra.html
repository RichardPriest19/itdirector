Microsoft Entra

Microsoft Entra – cloud-based identity, access & security platform (formerly Azure AD family)

Overview

Microsoft Entra is the unified family of identity, access, and security solutions that includes **Entra ID** (core identity service), **Entra ID Governance**, **Entra Verified ID** (decentralized identity), **Entra External ID** (B2B/B2C/CI), **Entra Permissions Management**, **Entra Internet Access**, **Entra Private Access**, **Entra ID Protection**, and the **Entra Suite** licensing tier. In March 2026 (post-Windows 11 24H2 / 25H2 and Microsoft 365 maturity), Entra is the cornerstone of Microsoft Zero Trust security, delivering **phishing-resistant MFA**, **Conditional Access**, **risk-based adaptive authentication**, **privileged identity management (PIM)**, **identity governance & lifecycle**, **global secure access** (SASE-like), **decentralized identity** (Verifiable Credentials), and seamless integration with Microsoft 365, Azure, Intune, Defender for Identity, Copilot, and thousands of third-party SaaS apps. It supports hybrid & multi-cloud identity, B2B/B2C collaboration, and compliance (GDPR, HIPAA, FedRAMP, ISO 27001).

Key Value: Microsoft Entra delivers unified, intelligent, Zero Trust identity and access management — enabling secure, password-less, adaptive access for users, devices, and apps across Microsoft 365, Azure, on-premises, and third-party environments while reducing risk, simplifying governance, and meeting global compliance requirements.

Core Capabilities (2026)

Entra ID (Core Identity)

  • Single sign-on (SSO), multi-factor authentication (MFA)
  • Phishing-resistant MFA (FIDO2, certificate-based, password-less)
  • Conditional Access policies & risk-based access

Entra ID Governance & PIM

  • Privileged Identity Management (just-in-time, approval-based access)
  • Access reviews, entitlement management & lifecycle workflows
  • Identity governance & compliance reporting

Entra External ID & Verified ID

  • B2B/B2C/CI collaboration & guest access
  • Decentralized identity (Verifiable Credentials)
  • External identity protection & fraud detection

Global Secure Access (Entra Internet/Private Access)

  • Secure web gateway & private access (SASE-like)
  • Zero Trust Network Access (ZTNA) for apps & resources
  • Integration with Microsoft Defender & Intune
