SCCM / MECM

Microsoft Systems/Endpoint Configuration Manager for on-premises endpoint management

Overview

Microsoft Systems Center Configuration Manager (SCCM), renamed to Microsoft Endpoint Configuration Manager (MECM), is Microsoft's on-premises endpoint management platform. While SCCM is considered legacy with Intune as the modern cloud-first solution, many enterprises still run SCCM for managing large Windows device fleets. Microsoft supports co-management where Intune and SCCM work together, allowing gradual migration to cloud-based management. SCCM remains critical in organizations with significant Windows deployments and on-premises infrastructure.

Key Transition: SCCM is transitioning to Intune, with co-management allowing both to work together during migration.

  • Windows Update management
  • Patch management
  • Application lifecycle management

Operating System Deployment

  • In-place upgrades

Hardware Inventory

  • Software license tracking
  • Reporting on assets

Configuration Management

  • Desired State Configuration (DSC)
  • Device compliance monitoring
  • Configuration baselines

Compliance and Reporting

  • Device compliance reporting
  • Software compliance
  • Audit and reporting

SCCM vs Intune

SCCM (On-Premises, Legacy)

  • Mature, feature-rich
  • Requires on-premises infrastructure
  • Best for managing Windows
  • Complex to manage

Intune (Cloud, Modern)

  • Cloud-based
  • Multi-platform (Windows, Mac, iOS, Android)
  • Easier to manage
  • Modern security capabilities

Co-Management

  • SCCM and Intune work together
  • Gradual migration to cloud
  • Use best of both platforms

IT Director Responsibilities

  • Plan SCCM/MECM deployment (if needed)
  • Design site hierarchy
  • Manage device collections
  • Oversee software and patch deployment

Migration Path to Intune

Co-Management Phase

  • Hybrid management with both SCCM and Intune
  • Gradual workload shift
  • Test and validate in Intune

Full Intune Migration

  • Complete move to cloud
  • Retire SCCM infrastructure
  • Benefits: cloud-based, modern, multi-platform
