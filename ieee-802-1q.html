IEEE 802.1Q

VLAN tagging standard – foundational protocol for Ethernet virtual LANs, trunking, and network segmentation

Overview

IEEE 802.1Q is the networking standard that defines VLAN tagging for Ethernet frames. It enables multiple virtual LANs to share the same physical infrastructure while maintaining logical separation, security, and traffic management.

Key Concepts

  • VLAN tagging (adds 4-byte 802.1Q tag to Ethernet frames)
  • Trunk ports vs Access ports
  • Native VLAN and untagged traffic handling
  • QinQ (802.1ad) for service provider double-tagging
  • Integration with QoS (802.1p priority) and MSTP

Best Practices 2026

  • Use 802.1Q for secure network segmentation in Zero Trust architectures
  • Implement proper VLAN pruning and avoid VLAN 1 as native VLAN
  • Combine with private VLANs (PVLAN) for enhanced security in multi-tenant environments
  • Document VLAN schemas and integrate with infrastructure-as-code (Terraform, Ansible)
  • Monitor trunk links and apply storm control / BPDU guard on access ports
