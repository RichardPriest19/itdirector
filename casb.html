CASB (Cloud Access Security Broker)

Security enforcement point between users and cloud service providers that provides visibility, compliance, data protection, and threat protection for SaaS applications

Overview

A Cloud Access Security Broker (CASB) sits between enterprise users and cloud applications (Microsoft 365, Google Workspace, Salesforce, etc.). It enables organizations to extend security policies into the cloud and gain visibility into shadow IT.

Core Functions (4 Pillars)

  • Visibility – Discovery of all cloud apps and usage
  • Compliance – Policy enforcement and regulatory reporting
  • Data Security – DLP and encryption for cloud data
  • Threat Protection – Detection of anomalous behavior and malware
