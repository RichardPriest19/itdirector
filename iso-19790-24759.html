ISO/IEC 19790 & 24759

ISO/IEC 19790 – security requirements for cryptographic modules & ISO/IEC 24759 – testing requirements (2012/2017 editions)

Overview

ISO/IEC 19790:2012 (with Corrigendum 1:2015) specifies the security requirements for cryptographic modules that provide cryptographic services (encryption, decryption, key generation, digital signatures, etc.). It defines four security levels (1–4) and is the international equivalent of FIPS 140-2/140-3. ISO/IEC 24759:2017 provides the derived test requirements and methods for validating conformance to ISO/IEC 19790. In March 2026, these standards remain the global benchmark for cryptographic module validation, forming the basis for FIPS 140-3 certification (U.S./Canada CMVP), Common Criteria evaluations, and compliance in regulated sectors (finance, government, healthcare, defense, critical infrastructure). They cover physical security, logical security, roles & authentication, key management, self-tests, and design assurance.

Key Value: ISO/IEC 19790 & 24759 provide rigorous, internationally recognized security and testing requirements for cryptographic modules — ensuring high assurance in encryption, key handling, and tamper resistance, reducing risk of compromise, enabling regulatory compliance (FIPS, Common Criteria), and building trust in cryptographic products and services.

Core Security Levels & Requirements (19790:2012 – 2026)

Security Levels (1–4)

  • Level 1 – basic security (production-grade components)
  • Level 2 – role-based authentication & tamper evidence
  • Level 3 – identity-based authentication & tamper resistance
  • Level 4 – highest security (tamper-proof, environmental protection)

Key Security Areas

  • Physical security (tamper detection/response)
  • Logical security (authentication, access control)
  • Roles, services & authentication
  • Key management (generation, entry/output, zeroization)
  • Self-tests & conditional tests
  • Design assurance & documentation

ISO/IEC 24759 – Testing Requirements

  • Derived test requirements for each 19790 assertion
  • Validation testing methods & evidence requirements
  • Used by accredited labs for certification (CMVP, Common Criteria)

Best Practices for ISO/IEC 19790 & 24759 Adoption (2026)
