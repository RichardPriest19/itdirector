Envoy Proxy

High-performance, programmable L7 proxy and communication bus widely used in service meshes and cloud-native architectures

Overview

Envoy was originally developed at Lyft and is now a CNCF graduated project. It powers many service meshes including Istio and is used as an API gateway and edge proxy.

Key Capabilities

  • Advanced load balancing and traffic management
  • Strong observability (stats, tracing, logging)
  • Dynamic configuration via xDS API
  • Extensive security features including mTLS
