GCFA

GIAC Certified Forensic Analyst – SANS/GIAC advanced certification for Windows and memory forensics investigation

Overview

The GIAC Certified Forensic Analyst (GCFA) is awarded by GIAC (Global Information Assurance Certification) and is based on the SANS FOR508 Advanced Incident Response and Threat Hunting course. It validates advanced skills in incident response, digital forensics, threat hunting, and memory analysis — equipping professionals to investigate complex intrusions, APT activity, and insider threats.

Key Value: GCFA is one of the most technically rigorous forensics certifications available — produced by SANS, the world’s leading cybersecurity training organisation, and covering advanced Windows artefact analysis, memory forensics, and threat hunting at a depth not matched by most comparable credentials.

Core Competency Areas

Advanced Windows Forensics

  • Analyse NTFS structures, MFT, $LogFile, $UsnJrnl, and Volume Shadow Copies
  • Examine Windows Registry forensic artefacts — UserAssist, ShimCache, Amcache, and BAM/DAM
  • Investigate Windows event logs — Security, System, Application, and PowerShell logs
  • Analyse prefetch, LNK files, jump lists, shellbags, and browser history artefacts

Memory Forensics

  • Acquire and analyse Windows memory images using Volatility and WinPmem
  • Identify injected code, hollowed processes, and malware resident in memory
  • Extract network connections, running processes, and loaded modules from memory
  • Analyse hibernation files and page files for forensic evidence

Incident Response & Threat Hunting

  • Apply the incident response lifecycle to APT and sophisticated threat actor investigations
  • Hunt for adversary persistence — scheduled tasks, services, registry run keys, and WMI subscriptions
  • Identify lateral movement artefacts — PsExec, WMI, RDP, and pass-the-hash evidence
  • Correlate artefacts across multiple systems to reconstruct multi-host intrusion timelines

Timeline Analysis

  • Build super-timelines using log2timeline/Plaso from multiple data sources
  • Apply timeline filtering and analysis to identify key investigative events
  • Correlate file system, registry, event log, and browser timelines for comprehensive incident reconstruction

Exam Details

  • 115 questions over 3 hours — open-book (notes permitted)
  • Minimum passing score: 73%
  • Based on SANS FOR508 Advanced Incident Response, Threat Hunting, and Digital Forensics course
  • Certification valid for 4 years; renewed via GIAC CPE credits or retaking the exam

IT Director Responsibilities

  • Deploy GCFA-certified analysts in senior incident response, threat hunting, and digital forensics roles
  • Use GCFA expertise for complex investigations involving APTs, ransomware, and insider threats
  • Integrate GCFA-level forensic capability into the incident response retainer or internal DFIR team
  • Recognise GCFA as an advanced-level qualification above foundational forensics certifications
