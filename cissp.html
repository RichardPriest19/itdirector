CISSP

Certified Information Systems Security Professional – ISC² gold-standard certification across 8 security domains

Overview

The Certified Information Systems Security Professional (CISSP) is awarded by ISC² (International Information System Security Certification Consortium) and is widely regarded as the gold standard in cybersecurity certifications. It validates deep technical and managerial knowledge across eight security domains and is required or preferred for senior security roles in governments, defence contractors, financial institutions, and technology companies worldwide.

Key Value: CISSP is the most widely recognised senior security certification globally — demonstrating breadth of knowledge across the full security domain spectrum and qualifying holders for senior architect, engineer, and leadership roles.

The 8 CISSP Domains (CISSP CBK 2024)

Security & Risk Management

  • Apply security governance principles, compliance, legal, and regulatory obligations
  • Conduct risk assessment, risk analysis, and risk treatment using recognised methodologies
  • Develop and maintain security policies, standards, procedures, and guidelines

Asset Security

  • Classify and manage data assets throughout their lifecycle
  • Apply data retention policies, secure disposal, and privacy protection controls

Security Architecture & Engineering

  • Design secure system architectures using security models, frameworks, and principles
  • Apply cryptography — symmetric, asymmetric, hashing, PKI, and key management
  • Evaluate and implement physical security controls for facilities and infrastructure

Communication & Network Security

  • Design and implement secure network architectures, segmentation, and protocols
  • Evaluate network attacks and countermeasures including firewalls, IDS/IPS, and VPNs

Identity & Access Management

  • Implement identification, authentication, authorisation, and accountability controls
  • Apply Zero Trust, federation, SSO, MFA, and privileged access management

Security Assessment & Testing

  • Design and validate assessment strategies — penetration testing, vulnerability assessment, code review
  • Analyse and interpret test outputs to drive security improvements

Security Operations

  • Conduct investigations, incident response, and digital forensics
  • Manage logging, monitoring, patch management, and configuration control

Software Development Security

  • Apply security throughout the SDLC including threat modelling, secure coding, and testing
  • Assess software security effectiveness and manage application vulnerabilities

Exam Details

  • CAT (Computerised Adaptive Testing) format — 125–175 questions over up to 4 hours
  • Prerequisite: 5 years of paid, full-time work experience in 2 or more of the 8 CISSP domains
  • Associate of ISC² designation available for those without the required experience
  • Renewal: 120 CPE credits over 3 years with annual AMF payment
  • DoD 8570/8140 approved — IASAE Level I, II, and III; IAM Level III

IT Director Responsibilities

  • Require CISSP for senior security architect, CISO, and principal security engineer appointments
  • Recognise CISSP as the technical-depth complement to management certifications such as CISM
  • Use CISSP holders to lead security architecture reviews, penetration test scoping, and control design
  • Ensure CISSP certification requirements are maintained for roles within DoD or government contract scope
