CCNA

Cisco Certified Network Associate – foundational vendor networking certification (200-301 exam)

Overview

CCNA (Cisco Certified Network Associate), exam code 200-301, is Cisco's flagship entry-to-mid-level networking certification. The current blueprint (v1.1, launched August 2020, still in force in March 2026 with continuous topic refresh) covers six domains: network fundamentals, network access, IP connectivity, IP services, security fundamentals, and automation/programmability. It validates skills in installing, configuring, operating, and troubleshooting enterprise networks (LAN/WAN, wireless, security basics, automation). In 2026, CCNA emphasizes modern networking: **SDN**, **Cisco DNA Center / Catalyst Center**, **automation (Python, Ansible, REST APIs)**, **wireless fundamentals**, **security (ACLs, VPN, threat defense)**, **cloud connectivity**, **IPv6**, and **network assurance**. It is the most popular vendor-specific networking certification globally, a prerequisite for CCNP Enterprise/Security/Data Center, and highly valued for network engineer, administrator, support, and junior architect roles in enterprise, service provider, and government environments.

Key Value: CCNA provides vendor-validated, practical networking expertise — covering modern enterprise networks, automation, security basics, and wireless — making it the gold standard entry credential for networking careers, enabling faster troubleshooting, better infrastructure design, and clear progression to CCNP/CCIE.

Core Exam Domains (200-301 v1.1 – 2026)

1. Network Fundamentals (20%)

  • OSI/TCP-IP models, cabling, IPv4/IPv6 addressing
  • Wireless fundamentals & AP modes
  • Cloud & virtualization concepts

2. Network Access (20%)

  • VLANs, trunking, EtherChannel
  • Spanning Tree Protocol (STP/RSTP)
  • Wireless LAN configuration & security

3. IP Connectivity (25%)

  • Routing concepts, static & dynamic routing (OSPF single-area)
  • IP addressing & subnetting (IPv4/IPv6)
  • Layer 3 redundancy (HSRP, VRRP)

4. IP Services (10%)

  • NAT, DHCP, SNMP, syslog, NTP
  • QoS concepts & classification/marking
  • Device management (SSH, Telnet, HTTPS)

5. Security Fundamentals (15%)

  • Access control lists (ACLs), VPN basics
  • Wireless security (WPA3, 802.1X)
  • Security program elements & threat defense

6. Automation and Programmability (10%)

  • Controller-based vs traditional networking
  • REST APIs, JSON, Python scripting basics
  • Configuration management (Ansible, Puppet, Chef)

Key Trends & Requirements in 2026

Exam & Certification

  • 120 minutes, ~100 questions, passing score ~825/1000
  • No prerequisites; valid 3 years, recertify via exam or CE credits
  • Continuous topic refresh (no major blueprint change since 2020)

Emerging Focus Areas

  • Automation/programmability weight increasing
  • Cloud connectivity & SD-WAN basics
  • Wireless security (WPA3, 802.1X) & IoT fundamentals
  • Zero Trust principles & basic threat defense

Career & Organizational Value

  • Entry-to-mid-level networking benchmark
  • Prerequisite for CCNP Enterprise/Security/Data Center
  • High demand in enterprise, service provider, government

Best Practices for CCNA Pursuit & Use (2026)

Certification Preparation

  • Study all 6 domains using official Cisco Learning Network

Application & Maintenance

  • Schedule via Pearson VUE (online or in-center)
