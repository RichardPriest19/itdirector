CompTIA Security+

CompTIA Security+ – vendor-neutral foundational cybersecurity certification and global entry-level security standard (SY0-701)

Overview

CompTIA Security+ is the most widely deployed vendor-neutral cybersecurity certification globally. It validates baseline security skills across threat management, cryptography, identity and access management, network security, and incident response. Security+ is DoD 8570/8140 approved and is commonly required as a minimum security qualification for IT and cybersecurity roles in government, defence, financial services, and enterprise environments.

Key Value: Security+ is the global baseline for cybersecurity practitioners — providing a vendor-neutral, broadly recognised foundation that qualifies professionals for a wide range of security roles and is the most common entry requirement for government and defence security positions.

Core Exam Domains (SY0-701)

General Security Concepts

  • Understand security controls, frameworks (NIST, CIS), and security principles
  • Apply cryptography concepts — symmetric, asymmetric, hashing, PKI, and digital signatures
  • Identify and categorise threat actors, attack vectors, and attack types

Threats, Vulnerabilities & Mitigations

  • Identify malware types, social engineering techniques, and application vulnerabilities
  • Perform vulnerability scanning and interpret results using CVSS and CVE
  • Apply security hardening techniques for operating systems, applications, and networks

Security Architecture

  • Design secure network architectures using DMZs, VLANs, and network segmentation
  • Apply cloud security concepts — shared responsibility, CSPM, and secure cloud configurations
  • Implement Zero Trust principles, microsegmentation, and software-defined perimeters

Security Operations

  • Monitor environments using SIEM, log analysis, and threat intelligence integration
  • Apply incident response procedures — preparation, identification, containment, and recovery
  • Conduct digital forensics basics — evidence preservation, chain of custody, and artefact analysis

Security Programme Management & Oversight

  • Apply risk management concepts — risk identification, assessment, and treatment
  • Manage third-party risk, vendor assessments, and supply chain security
  • Implement data privacy concepts and regulatory compliance requirements

Exam Details

  • Exam code: SY0-701 — up to 90 questions including performance-based questions over 90 minutes
  • Passing score: 750 out of 900
  • Recommended prerequisite: CompTIA Network+ and 2 years of IT experience with security focus
  • Certification valid for 3 years — renewed via CEU credits or higher-level CompTIA exam
  • DoD 8570/8140 approved at IAT Level II and CSSP Infrastructure Support

IT Director Responsibilities

  • Require Security+ as the minimum security qualification for all IT staff with security responsibilities
  • Use Security+ as the entry-level baseline before progressing staff to CySA+, PenTest+, or CASP+
  • Align Security+ coverage with security awareness training and organisational security policies
  • Include Security+ in job descriptions for service desk, network, and systems administration roles with security scope
