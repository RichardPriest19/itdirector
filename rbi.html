RBI (Remote Browser Isolation)

Security technology that isolates web browsing sessions in the cloud to protect endpoints from malicious websites and zero-day browser exploits

Overview

Remote Browser Isolation (RBI) executes web browsing sessions on a remote server or container in the cloud instead of on the user’s device. Only safe rendering information is sent back to the endpoint, effectively removing the attack surface from the browser.

Key Benefits

  • Protection against zero-day browser exploits and malware
  • Safe access to high-risk or untrusted websites
  • Reduced endpoint security burden
  • Strong complement to SWG and CASB solutions

Deployment Models

  • Full browser isolation
  • Selective isolation (only risky sites)
  • Integration with Secure Web Gateways (SWG)
