CCNP

Cisco Certified Network Professional – professional-level networking certification (Enterprise track dominant)

Overview

CCNP (Cisco Certified Network Professional) is Cisco’s professional-level certification track, validating advanced skills in planning, implementing, verifying, and troubleshooting enterprise networks. The most popular specialization is **CCNP Enterprise**, requiring core exam **350-401 ENCOR** + one concentration exam (e.g., 300-410 ENARSI, 300-415 ENSDWI, 300-420 ENSLD, 300-425 ENWLSD, 300-430 ENWLSI, 300-435 ENAUTO). In March 2026, CCNP Enterprise emphasizes modern enterprise networking: **SD-WAN**, **SD-Access**, **automation/programmability** (Python, Ansible, REST APIs, Cisco Catalyst Center), **wireless**, **security fundamentals**, **cloud connectivity**, **Zero Trust**, **network assurance**, and **dual-stack (IPv4/IPv6)**. It builds on CCNA, serves as a prerequisite for CCIE Enterprise Infrastructure/Wireless, and is highly valued for network engineer, architect, consultant, and senior support roles in enterprise, service provider, government, and large organizations.

Key Value: CCNP validates advanced, practical expertise in designing, implementing, and troubleshooting modern enterprise networks — with strong focus on automation, SD-WAN, wireless, security, and programmability — positioning professionals for senior roles, high salary premiums, and leadership in digital transformation and Zero Trust initiatives.

Core Exam Domains – ENCOR 350-401 (2026)

1. Architecture (15%)

  • Enterprise network design & high availability
  • SD-Access & SD-WAN concepts
  • Wireless network design & location services

2. Virtualization (10%)

  • Device virtualization technologies (VRF, VDC)
  • Network virtualization (VXLAN, LISP, EVPN)
  • Hypervisor & container concepts

3. Infrastructure (30%)

  • Layer 2 & Layer 3 technologies (switching, routing, BGP)
  • Wireless infrastructure & QoS
  • IP services (NTP, NAT, multicast, first-hop redundancy)

4. Network Assurance (10%)

  • Diagnostics & troubleshooting tools
  • Monitoring & telemetry (NetFlow, IP SLA, SPAN)
  • Network assurance with Cisco Catalyst Center

5. Security (20%)

  • Device access control & infrastructure security
  • REST API security & wireless security
  • Zero Trust concepts & basic threat defense

6. Automation (15%)

  • Programmability concepts (REST APIs, JSON, YANG)
  • Python scripting & Ansible basics
  • Cisco Catalyst Center & automation tools
