GCIH

GIAC Certified Incident Handler – SANS/GIAC certification for cybersecurity incident response and threat management

Overview

The GIAC Certified Incident Handler (GCIH) is awarded by GIAC and is based on the SANS SEC504 Hacker Tools, Techniques, and Incident Handling course. It validates knowledge of how attack techniques work and how to detect, respond to, and recover from security incidents — giving responders an attacker’s perspective to improve defensive effectiveness.

Key Value: GCIH is unique in combining offensive attack technique knowledge with incident response methodology — equipping responders to understand not just what happened, but how the attacker operated, enabling more effective containment and eradication.

Core Competency Areas

Incident Handling Process

  • Apply the PICERL incident handling process — preparation, identification, containment, eradication, recovery, and lessons learned
  • Operate within an IRT with defined scope, authority, and communication structures
  • Apply legal and evidence handling requirements during incident investigations
  • Produce post-incident reports with root cause analysis and remediation recommendations

Attacker Techniques & Detection

  • Understand and detect scanning, enumeration, and exploitation techniques
  • Identify malware types — viruses, worms, Trojans, ransomware, and rootkits — and their behavioural indicators
  • Detect web application attacks — SQL injection, XSS, and directory traversal — from logs and IDS alerts
  • Identify lateral movement techniques — pass-the-hash, Kerberoasting, and WMIC execution

Network & Host Forensics for Incident Response

  • Analyse network traffic and logs to reconstruct incident timelines
  • Examine Windows and Linux host artefacts to identify attacker footprints
  • Perform memory analysis for malware identification and process investigation
  • Use Volatility, Wireshark, and SIEM tools for evidence collection and analysis

Exam Details

  • 106 questions over 3 hours — open-book (notes permitted)
  • Minimum passing score: 73%
  • Based on SANS SEC504 Hacker Tools, Techniques, and Incident Handling
  • Certification valid for 4 years; renewed via GIAC CPE credits

IT Director Responsibilities

  • Require GCIH for security analysts, SOC team leads, and incident response professionals
  • Use GCIH holders to develop incident response playbooks informed by attacker technique knowledge
  • Integrate GCIH expertise into tabletop exercises and red team/blue team activities
  • Position GCIH as the progression from CSOC/ECIH towards advanced GCFA and threat hunting capability
