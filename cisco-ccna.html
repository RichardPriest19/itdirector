Cisco CCNA

Cisco Certified Network Associate – foundational networking certification covering routing, switching, and network fundamentals (200-301)

Overview

The Cisco Certified Network Associate (CCNA) is Cisco’s entry-to-intermediate level networking certification, validating knowledge of IP connectivity, network access, security fundamentals, automation, and programmability. The current single-track CCNA (200-301) replaced multiple legacy tracks in 2020 and covers a broad foundation of networking skills applicable across on-premises, hybrid, and cloud network environments.

Key Value: CCNA is the most widely recognised entry-level networking certification globally — setting the professional standard for network technicians, junior network engineers, and IT administrators working with Cisco infrastructure.

Core Exam Domains (200-301)

Network Fundamentals

  • Understand OSI and TCP/IP models, network topologies, and physical layer concepts
  • Configure and troubleshoot IPv4 and IPv6 addressing, subnetting, and VLSM
  • Explain switching concepts — MAC address tables, VLANs, and spanning tree protocol (STP)

Network Access

  • Configure and verify inter-VLAN routing using Layer 3 switches and router-on-a-stick
  • Implement EtherChannel, RSTP, and port security on Cisco switches
  • Configure wireless LAN controllers and understand wireless security protocols

IP Connectivity & Services

  • Configure OSPFv2 for single-area IPv4 routing
  • Implement DHCP, DNS, and NAT on Cisco IOS routers
  • Understand QoS concepts and first-hop redundancy protocols (HSRP)

Security Fundamentals

  • Configure ACLs, port security, DHCP snooping, and dynamic ARP inspection
  • Implement site-to-site and remote-access VPNs on Cisco routers and firewalls
  • Apply AAA concepts and device access control using TACACS+ and RADIUS

Automation & Programmability

  • Understand network automation, SDN, and controller-based networking concepts
  • Work with REST APIs, JSON, YANG, and network programmability tools
  • Apply configuration management tools — Ansible, Puppet, Chef — in network contexts

Exam Details

  • Exam code: 200-301 — approximately 100 questions including multiple-choice, drag-and-drop, and simulation
  • 120-minute exam duration; no formal prerequisites
  • Certification valid for 3 years; renewed by passing a professional-level exam, CCNA recertification, or Cisco CE credits
  • Foundation for Cisco professional tracks: CCNP Enterprise, CCNP Security, and DevNet

IT Director Responsibilities

  • Require CCNA for network technician, junior network engineer, and desktop support roles managing network infrastructure
  • Use CCNA as the baseline qualification for Cisco-heavy environments before advancing to CCNP
  • Align CCNA training with network automation and SDN upskilling to meet 2026 infrastructure demands
  • Consider CCNA as a prerequisite for progressing to CCNP Security or Cisco CyberOps certifications
