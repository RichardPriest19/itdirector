Figma

Collaborative interface design tool used for UI/UX design, prototyping, wireframing, and design systems

Overview

Figma is the leading cloud-based design and prototyping platform. It enables real-time collaboration, version history, and seamless handoff from design to development, making it the standard tool for product design teams worldwide.

Key Capabilities

  • Real-time multiplayer editing and commenting
  • Advanced prototyping and interactive components
  • Design Systems and component libraries
  • Auto Layout, Variants, and Variables
  • Dev Mode for developer handoff and inspection
  • FigJam for collaborative whiteboarding and workshops

Enterprise Features (2026)

  • SSO, SCIM, and advanced admin controls
  • Private libraries and organization-wide design systems
  • Usage analytics and governance tools
