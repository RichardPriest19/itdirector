EC2

Amazon Elastic Compute Cloud – secure, resizable compute capacity in the cloud

Overview

Amazon Elastic Compute Cloud (EC2) is AWS's flagship virtual server service that provides resizable compute capacity in the cloud. In March 2026, EC2 remains the most widely used compute service globally, with massive scale and continuous innovation. Key 2026 highlights include **Graviton5** (Arm-based) instances leading price/performance, **Trainium2/Inferentia3** accelerators powering frontier AI training/inference, **confidential computing** maturity (Nitro Enclaves + AMD SEV-SNP), **Spot Fleets** & **Savings Plans** optimization as FinOps standard, **Capacity Blocks** for guaranteed GPU capacity, **EC2 Auto Scaling** enhancements for bursty AI workloads, **edge/local zones** expansion for low-latency, and **zero-trust instance posture** (IMDSv2 mandatory, workload identity federation). EC2 supports virtually every workload: web apps, batch processing, AI/ML training/inference, databases, gaming, HPC, and containers (ECS/EKS).

Key Value: EC2 provides secure, elastic, global compute capacity with unmatched instance variety, performance, cost optimization tools, and deep AWS integration — enabling organizations to run any workload with high availability, rapid scaling, strong security, and — with Graviton + AI accelerators — achieve best-in-class price/performance for modern cloud-native and AI-driven applications.

Core Instance Types & Capabilities (2026)

General Purpose & Compute Optimized

  • Graviton5 (Arm64) – best price/performance for most workloads
  • M8g / C8g – latest Graviton generations
  • M7i-flex / C7i – Intel/AMD for legacy compatibility

AI/ML & Accelerated Compute

  • Trainium2 / Inferentia3 – purpose-built AI accelerators
  • P5 / P6 instances – NVIDIA H200 & Blackwell GPUs
  • Trn2 / Inf3 – cost-effective training/inference at scale

Memory & Storage Optimized

  • R8g / X8g – high-memory Graviton
  • I4i / Im4gn – high IOPS NVMe instances
  • High I/O & ultra-fast local NVMe

Security & Specialized

  • Nitro Enclaves + AMD SEV-SNP – confidential computing
  • Mac instances (Apple silicon) for iOS/macOS workloads
  • Hpc7g / Hpc7a – high-performance computing

Security & Governance
